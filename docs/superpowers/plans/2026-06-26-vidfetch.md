# VidFetch Implementation Plan (Updated)

**Goal:** Static web app with **2 functional features**: Download YouTube videos + History with auto-cleanup.

**Scope (user-approved):**
- ✅ Download — paste URL → fetch preview + all format options → open download
- ✅ History — list downloads (thumbnail + link + copy button), auto-delete after 1 week via localStorage
- ❌ Fix MP4, Audio Only, Batch Queue, Settings — REMOVED

**Sidebar:** Download + History only (plus logo + storage bar visual).

**Tech:** Vanilla HTML/CSS/JS, Tabler Icons, dark metallic silver theme. Deploy to Cloudflare Pages.

**Endpoint:** `GET https://api.synoxcloud.xyz/download/all-in-one?url=<URL>` (locked to YouTube)

**JSON mapping (verified):**
- `result.data.title` → title
- `result.data.thumbnail` → thumbnail URL
- `result.data.duration` → seconds (number)
- `result.data.author` → channel name (may be empty)
- `result.data.medias[]` → format list (.label, .type, .ext, .url, .bitrate, .width, .height)
