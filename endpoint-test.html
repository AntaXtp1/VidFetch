📚 Cara baca hasil:
• Status 200 + ada response = ✅ endpoint jalan & CORS-nya kebuka (browser bisa akses langsung)
• "Failed to fetch" / TypeError = ❌ CORS ditutup ATAU server down → butuh proxy server
• Status 5xx / 4xx = endpoint bermasalah di sisi server
• Timeout = server lambat/tidak respon
